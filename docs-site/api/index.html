HTTP endpoint index

Complete endpoint map for MemState API surfaces, including auth behavior and module ownership.

Auth summary

  • When MEMSTATE_API_KEY is set, protected JSON routes require X-API-Key or Authorization: Bearer.
  • /health, /health/graph, /health/falkordb, and static /ui remain unauthenticated.
