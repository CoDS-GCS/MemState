Power-user tools

When you need more than the happy path.

Beyond the stable client API, MemState exposes deeper operations for debugging, migrations, and internal tools. Those surfaces trade some stability guarantees for precision and visibility.

Who should use them

Engineers maintaining the deployment, data scientists validating corpus quality, and automated tests that need deterministic hooks. Product features aimed at end users should stay on the client API whenever possible.
