Build on MemState with clear contracts.

These pages track how the reference implementation behaves today: HTTP routes, storage layout, reasoner hooks, and operational limits. The product contract is fixed: agents emit observations and ask for context; MemState owns ingest structure, revise, and forget. Each technical page has a shorter product twin focused on outcomes and adoption.

Implementation status

MemState is an open reference prototype. Behavior matches the codebase in src/memstate; where the implementation is still catching up to the full research model, the technical pages call that out in an Implementation notes section at the bottom (injected on each developer page).
